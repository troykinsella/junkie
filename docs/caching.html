Module: Resolver:caching

Resolver:caching

Searches the Component#store for a cached instance of a previously resolved component. If one is found, it is resolved and the resolution is marked as done. Otherwise, the next resolvers are called, and when they complete, the resulting resolved instance is cached such that subsequent resolves will return this instance.
