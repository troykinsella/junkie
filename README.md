# larry [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Larry is a regular kinda build server.


## Install

```sh
$ npm install --save larry
```


## Usage

```js
var larry = require('larry');

larry('Rainbow');
```
